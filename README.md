# Assignment-6

Project Overview

This is a personal portfolio website built with HTML, CSS, and JavaScript to showcase my work as a front-end web developer and visual designer. The website highlights featured works, demonstrates responsive design skills, and includes an interactive contact form with validation and a modal confirmation popup.

The project emphasizes clean design, accessibility, and interactivity, giving visitors a seamless browsing experience across devices.

Features

Responsive Homepage – With a navigation bar and portfolio introduction
Interactive Featured Work Gallery – Highlights three main practices with hover effects and animations

Welcome Button Popup – A friendly interactive message for users
Contact Section – Accessible from homepage and contact page
Contact Page – Includes:
Name, email, and message fields
Real-time validation with instant error messages
Modal confirmation on successful submission
Clean & Modular Code Structure – HTML, CSS, and JS files separated for easy maintenance

Technologies Used

HTML5 – Semantic markup for better accessibility and SEO
CSS3 – Styling, responsive layouts, and animations
JavaScript (ES6) – DOM manipulation, form validation, interactivity, and modals
Tools – VSCode, Chrome DevTools, Git & GitHub
W3C Validator – Ensures HTML and CSS follow web standards
